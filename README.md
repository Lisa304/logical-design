---
- 鳥會往下掉,用按鍵使其向上
- 多個障礙物
- 隨機出現障礙物，會有不同位置.大小.
- 不同關卡
- 計分制
- 記生命(生命歸零就game over)
- 闖過過關和碰到障礙物都會發出聲音
